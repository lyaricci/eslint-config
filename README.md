# ESlint config for NodeJS

## What is included?

- Standard config base;
- Prettier;
- Typescript support

## How to setup

1. Install dependencies

```bash
npm i -D @nathalyaricci/eslint-config
```

2. Extend the config in a `.eslintrc.js` or `.eslintrc.json` file

```bash
# .eslintrc.js

module.exports = {
  extends: ["@nathalyaricci/eslint-config"],
}
```

```bash
# .eslintrc.json

{
  extends: ["@nathalyaricci/eslint-config"],
}
```
